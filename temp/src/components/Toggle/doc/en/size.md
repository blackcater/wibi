### Size

The `size` property sets the switch size, and the value includes: `lg`、`md`、`sm`.

<!--start-code-->

```vue
<template>
  <div>
    <Toggle style="margin-bottom: 8px;" size="lg" />
    <Toggle style="margin-bottom: 8px;" size="md" />
    <Toggle style="margin-bottom: 8px;" size="sm" />
  </div>
</template>
```

<!--end-code-->
